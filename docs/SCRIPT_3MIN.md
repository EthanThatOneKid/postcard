# PostcardHQ Demo Script (3:00)

### 1. Introduction (0:00 - 0:10)

- **Point:** "Hi, we're PostcardHQ. We built a tool to verify social media posts and trace them back to their origin."

### 2. The Problem (0:10 - 0:35)

- **Point:** "Most viral information is shared as screenshots. These images strip away metadata, links, and context, making it impossible to know if a post is real or edited."
- **The Gap:** "Once a post goes viral, the link to the original source is often lost."

### 3. The Solution (0:35 - 1:00)

- **Point:** "Postcard is an open-source API that finds and verifies that source. It’s an alternative to proprietary tools like Grok but focused on technical verification."
- **Tech:** "We use a 4-stage pipeline to ingest the post, find the original version, and identify any differences between the viral screenshot and the live metadata."
